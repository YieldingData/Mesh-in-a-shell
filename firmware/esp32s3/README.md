# ESP32-S3 Firmware

This directory contains the Arduino or ESP-IDF code that runs on the XIAO ESP32-S3. It handles:

- LoRa messaging
- GPS parsing and hourly push
- Button input
- e-paper UI and boot mini-game
- Power control for the Orange Pi

## Dependencies

- GxEPD2
- Adafruit_FT6206
- LoRa library (SX1262)
